# UX Flow

## 기본 사용자 흐름

1. Hero에서 서비스 이해
2. 문제 섹션에서 내 상황과 연결
3. 해결 방식 확인
4. 서비스 종류와 작업 범위 확인
5. Before & After로 결과 신뢰
6. 가격과 진행 과정 확인
7. 후기 확인
8. FAQ로 불안 해소
9. 예약 CTA 클릭

## 예약 전환 흐름

Hero CTA -> 가격/일정 확인 -> FAQ -> 예약 상담

## 모바일 핵심 흐름

- 상단 메뉴는 간단히 유지한다.
- 하단 고정 예약 버튼을 제공한다.
- 가격, FAQ, 예약 버튼까지 스크롤 흐름을 짧게 유지한다.
