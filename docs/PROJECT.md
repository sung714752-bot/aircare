# AirCare 프로젝트 개요

## 프로젝트명

AirCare (가칭)

## 프로젝트 형태

에어컨 청소 전문 반응형 원페이지 홈페이지

## 목적

사용자가 에어컨 청소 필요성을 빠르게 이해하고, 신뢰를 느낀 뒤 예약 또는 문의까지 자연스럽게 이동하도록 만든다.

## 핵심 메시지

깨끗한 바람은 깨끗한 에어컨에서 시작됩니다.

## 주요 목표

- 첫 화면에서 서비스와 장점을 3초 안에 이해시킨다.
- 가격, 작업 범위, 진행 과정, 후기, FAQ를 명확히 제공한다.
- 모바일 사용자가 빠르게 예약할 수 있게 한다.
- 실제 작업 사진과 Before/After 비교로 신뢰를 만든다.

## 타깃 고객

- 여름철 에어컨 청소가 필요한 가정
- 냄새, 곰팡이, 먼지 문제를 겪는 사용자
- 아이, 반려동물, 알레르기 등 실내 공기질에 민감한 고객
- 사무실, 소형 매장, 원룸, 오피스텔 사용자

## 참고 벤치마킹 방향

미소 같은 홈서비스 홈페이지의 장점인 쉬운 흐름, 명확한 CTA, 신뢰 요소 배치를 참고하되 에어컨 청소 서비스에 맞는 차별화 기능을 추가한다.
