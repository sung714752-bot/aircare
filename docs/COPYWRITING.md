# Copywriting

문구만 빠르게 수정하기 위한 문서입니다. 실제 화면 문구는 `CONTENT.md`를 기준으로 하되, 이 문서는 대체 문구와 톤을 관리합니다.

## 톤

- 전문적이지만 어렵지 않게
- 불안을 과하게 자극하지 않기
- 가격, 범위, 과정은 투명하게
- CTA는 짧고 행동 중심으로

## Hero 대체 문구

- 에어컨 속 곰팡이와 냄새, 전문 세척으로 깨끗하게
- 우리 집 공기를 바꾸는 에어컨 완전분해 세척
- 보이지 않는 에어컨 속 오염까지 꼼꼼하게

## CTA 대체 문구

- 예약 상담하기
- 가격 확인하기
- 작업 사례 보기
- 우리 지역 가능 여부 확인

## 신뢰 문구

- 작업 전후를 사진으로 확인하세요.
- 에어컨 종류별 작업 범위를 투명하게 안내합니다.
- 상담부터 작업 완료까지 친절하게 안내합니다.
