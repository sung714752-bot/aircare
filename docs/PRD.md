# PRD

## 문제 정의

에어컨 청소 고객은 보통 아래를 궁금해한다.

- 이게 왜 필요한가?
- 곰팡이와 냄새가 정말 해결되는가?
- 가격은 얼마인가?
- 어떤 방식으로 진행되는가?
- 우리 지역도 가능한가?
- 예약은 어떻게 하는가?

## 해결 방식

홈페이지는 고객의 심리 순서대로 정보를 제공한다.

1. 문제 인식
2. 해결책 제시
3. 서비스 범위 설명
4. 실제 사례와 후기 제공
5. 가격과 절차 안내
6. FAQ로 불안 해소
7. 예약 CTA 제공

## 핵심 기능

- 반응형 원페이지 레이아웃
- Header 내 주요 섹션 앵커 이동
- Hero CTA 버튼
- Before & After 비교 영역
- 가격 안내 또는 예상 가격 계산기
- FAQ 아코디언
- 후기 카드와 필터
- 모바일 하단 고정 예약 버튼
- 부드러운 스크롤

## 제외 기능

초기 버전에서는 실제 결제, 실제 예약 DB, 로그인, 관리자 페이지, 실시간 캘린더 연동은 제외한다.

## 성공 기준

- 모바일에서 주요 CTA가 항상 찾기 쉽다.
- 사용자가 가격, 작업 범위, 소요 시간, 예약 방법을 한 화면 흐름 안에서 이해한다.
- Lighthouse 기준 성능, 접근성, SEO 점수를 개선 가능한 구조로 만든다.
