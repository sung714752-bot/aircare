# Design System

## 디자인 키워드

깨끗함, 신뢰감, 전문성, 시원함, 친절함

## 색상

- Primary: 선명한 블루 계열, 예약/주요 CTA에 사용
- Secondary: 청록 또는 민트 계열, 청결함과 공기감을 표현
- Text: 진한 네이비 또는 차콜
- Muted Text: 회색 계열
- Background: 흰색과 아주 옅은 쿨 그레이
- Warning/Notice: 가격, 마감, 예약 안내에 제한적으로 사용

## 타이포그래피

- 한국어 가독성이 좋은 sans-serif 사용
- Hero 제목은 강하게, 본문은 편하게 읽히는 크기 사용
- 모바일에서는 제목 줄바꿈을 우선 고려한다.

## 레이아웃

- 모바일 우선
- 섹션 간 여백은 넉넉하게 두되 CTA 흐름이 끊기지 않게 한다.
- 카드 UI는 후기, 가격, 서비스 항목처럼 반복 정보에만 사용한다.
- 페이지 섹션은 전체 폭 흐름을 유지한다.

## 버튼

- Primary CTA: 예약하기, 상담하기
- Secondary CTA: 가격 보기, 작업 사례 보기
- 버튼 문구는 행동 중심으로 작성한다.
- 모바일 터치 영역은 충분히 크게 둔다.

## 이미지

- 실제 작업 사진, Before/After, 장비, 청소 과정 중심
- 어둡거나 모호한 분위기 사진은 피한다.
- 고객이 결과를 확인할 수 있는 선명한 사진을 우선한다.

## 애니메이션

- 과하지 않은 fade/slide 정도만 사용한다.
- 예약 CTA와 FAQ 조작은 즉각 반응해야 한다.
