# Prompts

## 섹션 작업 시작

현재 단계는 `[섹션명]`입니다. `docs/PROJECT.md`, `docs/PRD.md`, `docs/DESIGN_SYSTEM.md`, `docs/CONTENT.md`를 참고해서 먼저 구조를 제안하고, 필요한 요소를 체크리스트로 정리한 뒤 구현해줘.

## Hero

기획서를 바탕으로 Hero Section을 만들어줘. 제목, 보조문구, CTA 버튼 2개, 실제 서비스 이미지를 넣고 모바일에서 자연스럽게 보이게 해줘.

## FAQ

FAQ Accordion을 만들어줘. 키보드 접근성과 `aria-expanded` 상태 업데이트를 포함해줘.

## 반응형 검토

현재 페이지를 모바일, 태블릿, 데스크톱 기준으로 점검하고 텍스트 겹침, 버튼 잘림, 섹션 여백 문제를 수정해줘.

## 마감 점검

`docs/CHECKLIST.md`, `docs/TEST_PLAN.md`, `docs/ACCESSIBILITY.md` 기준으로 최종 점검하고 수정해줘.
