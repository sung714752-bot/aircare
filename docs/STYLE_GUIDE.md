# Style Guide

## 파일명

- 소문자와 하이픈 사용
- 예: `main.css`, `price-calculator.js`

## HTML

- semantic 태그 사용
- 섹션에는 명확한 `id` 부여
- 이미지에는 의미 있는 `alt` 작성

## CSS

- CSS 변수 사용
- class 이름은 kebab-case
- 모바일 우선 작성
- 반복되는 값은 토큰화

## JavaScript

- Vanilla JavaScript 기준
- 함수명은 동작이 드러나게 작성
- DOM 선택자는 상단에 모아 관리
- 접근성 속성(`aria-expanded` 등)을 함께 업데이트

## 주석

복잡한 상호작용이나 의도 설명이 필요한 곳에만 짧게 작성한다.
