# Image Guidelines

## 원칙

- 실제 작업 사진을 우선 사용한다.
- Before/After 비교가 가능한 이미지를 확보한다.
- 어둡거나 흐린 이미지는 피한다.

## 권장 형식

- WebP 우선
- 원본 보관이 필요하면 `public/assets/originals/`에 별도 보관
- 실제 사용 이미지는 용량 최적화 후 사용

## 파일명 규칙

- 소문자, 하이픈 사용
- 예: `wall-ac-before.webp`
- 예: `system-ac-cleaning-process.webp`

## 필수 이미지 후보

- Hero 대표 이미지
- 벽걸이 에어컨 작업 전/후
- 스탠드 에어컨 작업 전/후
- 작업 장비 사진
- 작업자 또는 작업 과정 사진
