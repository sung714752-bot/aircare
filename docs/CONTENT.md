## Problem

- 에어컨을 켤 때 냄새가 난다.
- 송풍구 안쪽에 검은 곰팡이가 보인다.
- 아이와 가족의 실내 공기가 걱정된다.
- 직접 청소하기 어렵고 어디까지 분해해야 할지 모르겠다.

## Solution

- 전문 장비로 내부 오염을 분리 세척한다.
- 작업 전후 사진으로 결과를 확인한다.
- 벽걸이, 스탠드, 시스템 에어컨별 작업 범위를 안내한다.
