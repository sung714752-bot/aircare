# Accessibility

## 기준

- 키보드만으로 메뉴, FAQ, CTA를 사용할 수 있어야 한다.
- 포커스 스타일을 제거하지 않는다.
- 텍스트와 배경의 대비를 충분히 확보한다.
- 아이콘만 있는 버튼에는 접근 가능한 이름을 제공한다.
- 이미지에는 의미 있는 대체 텍스트를 작성한다.

## 확인 항목

- [ ] Tab 순서가 자연스러운가?
- [ ] 모바일 메뉴 열림 상태가 스크린리더에 전달되는가?
- [ ] FAQ 버튼의 `aria-expanded`가 변경되는가?
- [ ] CTA 버튼 텍스트가 명확한가?
- [ ] 이미지 alt가 장식/정보 이미지를 구분하는가?
