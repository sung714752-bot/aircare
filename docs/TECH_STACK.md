# Tech Stack

## 초기 권장 스택

- HTML5
- CSS3
- Vanilla JavaScript

## 기준

초기 버전은 빠르게 이해하고 수정할 수 있는 정적 사이트로 만든다. 프레임워크는 프로젝트 규모가 커지거나 컴포넌트 재사용 요구가 커질 때 도입한다.

## 브라우저 지원

- 최신 Chrome
- 최신 Edge
- 모바일 Safari 기준 레이아웃 고려
